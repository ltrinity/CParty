Requirements\+: A compiler that supports C++11 standard (tested with g++ version 4.\+7.\+2 or higher) and CMake version 3.\+1 or greater. ~\newline


\href{https://cmake.org/install/}{\texttt{ CMake}} version 3.\+1 or greater must be installed in a way that HFold can find it. ~\newline
 To test if your Mac or Linux system already has CMake, you can type into a terminal\+: ~\newline
 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ -\/-\/version}

\end{DoxyCode}
 If it does not print a cmake version greater than or equal to 3.\+1, you will have to install CMake depending on your operating system.\hypertarget{md__c_1_2_users_2luket_2_desktop_2testing___c_party_2_c_party_2_r_e_a_d_m_e_autotoc_md5}{}\doxysubsubsection{\texorpdfstring{Mac\+:}{Mac:}}\label{md__c_1_2_users_2luket_2_desktop_2testing___c_party_2_c_party_2_r_e_a_d_m_e_autotoc_md5}
